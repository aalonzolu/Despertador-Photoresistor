# Despertador-Photoresistor
Despertador que reproduce una cancion cuando detecta luz.

### Ejemplo de uso
```sh
$ git clone https://github.com/andyalonzo/Despertador-Photoresistor.git
$ cd Despertador-Photoresistor
$ node index.js
```
